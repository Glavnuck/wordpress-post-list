This plugin generates tag cloud and posts list ordered by categories. The list may be placed in any part of wordpress page contents.

== Description ==

= Benefits of WP Post List =
* It may display full tag cloud and well formed post list in any place of page/post contents
* It doesn't require any theme modifications or any kind of coding to use it
* For long post lists WPPL supports cool smooth scrolling feature
* It generates well-formed HTML code with CSS classes

= Settings / Options =
* Set title for tag cloud and post list
* Change replace strings for tag cloud and post list if needed
* Toggle smooth scrolling
* Delete settings from the database

<a href="http://paradigm.ru/archive">Post list demo</a>.

WPPL is based on <a href="http://www.jenst.se/2008/03/30/wp-simple-sitemap">WP Simple Sitemap</a> (by Jens T&ouml;rnell) which became some kind of tutorial in WP API for me.

== Installation ==

Download: <a href="http://paradigm.ru/wp-postlist">WP Post List</a>

1. Upload the FOLDER 'wp-postlist' to the '/wp-content/plugins/'
2. Activate the plugin 'WP Post List' through the 'Plugins' menu in admin
3. Go to 'Options' or 'Settings' and then 'Simple Sitemap' to change the options

= Usage =

1. Choose the page or post where you want WP Post List to be displayed.
2. Place the <code>[postlist]</code> string in any part of the page and save/publish your page.

That's all. The replace string may variate through plugin options.
